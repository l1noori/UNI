Opdracht 2 Datastructuren
Author: Noah van Laarhoven

The program can be called with ./main <filename>

This program has the following functions:

* exp <expression>
Read the expression in prefix-form

* dot <filename>
Saves the given expression using the 'exp' function to the given file, in DOT notation

* eval <value>
Evaluates the expression

* diff
Differentiates the expression (to x)

* simp
Simplifies the expression

EXTRA:

* print
Prints the current expression in infix-form
